# Nexora 0.1.0

发布日期：2026-07-17

Nexora 的首个 early-alpha 源码版本。

## Added

- 提供同名 `nexora` 框架库与 CLI，支持 `create`、`init`、macOS `build`、`doctor` 和
  workspace `lint`。— 处理人：[@Jixiangup](https://github.com/Jixiangup)
- 提供 single/workspace 项目模板和自动写入的框架开发 Skills。— 处理人：
  [@Jixiangup](https://github.com/Jixiangup)
- 提供 GPUI Application Shell、Feature/Window 自动注册、侧边导航、登录门禁、设置窗口、
  强类型路由与生命周期。— 处理人：[@Jixiangup](https://github.com/Jixiangup)
- 提供 OIDC、用户、角色、权限和可重试初始化接口。— 处理人：
  [@Jixiangup](https://github.com/Jixiangup)

## Fixed

- 生成项目使用固定到对应版本 tag 的 Nexora Git 依赖。— 处理人：
  [@Jixiangup](https://github.com/Jixiangup)
- 修正桌面标签右键菜单的窗口上下文与关闭目标。— 处理人：
  [@Jixiangup](https://github.com/Jixiangup)
