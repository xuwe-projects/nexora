## 新增

- 登录页、侧边导航顶部和窗口品牌信息现在统一使用应用名称、版本与自定义 Logo。
- 账号功能默认提供用户管理、角色与权限管理页面，并按“访问控制”分组显示导航。
- 登录失败通过通知反馈；服务端返回 request ID 时可以直接复制，方便排查问题。

## 体验优化

- 用户头像区域展示正确的 display name，不再使用外部 identity ID 代替名称。
- 侧边导航分组之间使用位于下一组标题上方的分隔线，分组关系更加清晰。
- 自定义侧边导航顶部或底部内容时，仍保留统一的悬停反馈、间距和分隔线。

## 开发体验

- 桌面认证配置、登录会话和 Account HTTP 客户端统一从 `nexora::desktop` 使用。
- 生成项目附带 Logo 资源、带注释的配置、项目 Rules 和轻量 Feature 组件化 Skills。
