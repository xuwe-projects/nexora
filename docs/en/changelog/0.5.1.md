# Nexora 0.5.1

Release date: 2026-07-18

This patch fixes the button-disabled mapping in `FormDialog::submit_disabled`. Public APIs,
configuration, database migrations, and dependency revisions remain compatible with 0.5.0, so the
release increments PATCH under Semantic Versioning.

## Fixed

- Cancel is disabled only while a submission is in progress. Submit is disabled while submitting or
  when the caller sets `submit_disabled(true)`. Invalid forms can now still be cancelled without
  allowing submission. — Owner: [@Jixiangup](https://github.com/Jixiangup)
- Added a GPUI click regression test and a source-contract test to prevent the two conditions from
  being swapped again. — Owner: [@Jixiangup](https://github.com/Jixiangup)

## Compatibility

There are no breaking changes and no caller migration is required. GPUI, gpui-component, ZITADEL,
the HTTP API, and the database schema are unchanged.

## Upgrade

Update the Nexora tag from `v0.5.0` to `v0.5.1` and refresh the lockfile. The published `v0.5.0` tag
remains unchanged. No database migration or configuration change is required.

## Validation

- `cargo test --workspace --all-targets`
